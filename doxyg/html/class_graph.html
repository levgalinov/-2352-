Detailed Description

A class representing a directed graph for finding maximum flow using the Edmonds-Karp algorithm.

This class implements a graph structure with capacity and flow matrices, as well as the Edmonds-Karp algorithm for finding maximum flow in the network. The graph is directed, and for each edge the capacity and flow are stored.
